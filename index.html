colors

every color scheme has 5 colors:

  • white: ... white, duh
  • gray: whatever gray is used for the body text
  • primary: the main accent color, usually very bright
  • secondary: the secondary accent color, usually very bright
  • light: a lighter accent color suited for backgrounds

all colors are displayed in that order.

helper classes

every color scheme has the following helper classes:

  • background-$COLOR
  • text-$COLOR
  • button-background-$COLOR
  • button-text-$COLOR

... where $COLOR is the name of the color you wish to use. they will work on all elements, but will provide no other styling.

color schemes

to use a color scheme, add the color scheme name to the body element's class. you don't have to do anything else!

if you're using the default color scheme, you don't have to do anything at all.
